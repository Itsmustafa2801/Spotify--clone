# Spotify Clone 🎵

A web-based Spotify clone that allows users to play songs, control playback, and browse music.

## 🚀 Features
- Play/Pause songs
- Next/Previous track controls
- Volume control
- Song progress bar
- Song list with thumbnails

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
